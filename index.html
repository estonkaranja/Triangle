Definition.

A triangle is a closed figure made up of three line segments.

Types of Triangles

there are different types of triangles. these include:

  2. Triangles based on sides.

Types of Triangles.

  1. Equilateral Triangle
  2. Scalene Triangle
  3. Isosceles Triangle

Featured Triangles.

  1. Equilateral Triangle: A triangle having all the three sides of equal length is an equilateral triangle. E.g length = 2, width = 2, height = 2.
  2. Scalene Triangle: A triangle having three sides of different lengths is called a scalene triangle. E.g length = 4, width = 5, height = 6.
  3. Isosceles Triangle: A triangle having two sides of equal length is an Isosceles triangle. E.g length = 2, width = 2, height = 3.
